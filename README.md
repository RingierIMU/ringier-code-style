# ringier-code-style
RingerSA PHP Code Style

## Installation

```bash
composer require ringierimu/ringier-code-style --dev
```

## Usage

### Fix code style
```bash
vendor/bin/ringier-code-style fix ...files...
```

### Create initial .styleci.yml config
```bash
vendor/bin/ringier-code-style styleci:init
```
